# 그리디 알고리즘(Greedy algorithm)

## 그리디 알고리즘이란?
다음을 생각하지 않고 각 단계에서의 가장 최선의 선택을 하는 알고리즘

## 그리디 알고리즘의 특징
- 다음 단계를 생각하지 않고 각 단계에서의 최선의 선택을 함

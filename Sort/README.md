# 정렬 알고리즘(Sort algorithm)

## 정렬 알고리즘 종류
1. [버블 정렬](#버블-정렬)
2. 삽입 정렬


### 버블 정렬
> 1. 이중반복문을 통해 서로 붙어있는 인덱스의 크기를 비교
> 2. 크기가 크거나 작으면 서로 바꿈  

<b>시간 복잡도 : 이중반복문을 사용하므로 시간복잡도는 O(n²)이다.  </b>

* 예시  

First Pass:  
( 5 1 4 2 8 ) –> ( 1 5 4 2 8 )  
( 1 5 4 2 8 ) –>  ( 1 4 5 2 8 )  
( 1 4 5 2 8 ) –>  ( 1 4 2 5 8 )  
( 1 4 2 5 8 ) –> ( 1 4 2 5 8 )

Second Pass:  
( 1 4 2 5 8 ) –> ( 1 4 2 5 8 )  
( 1 4 2 5 8 ) –> ( 1 2 4 5 8 )  
( 1 2 4 5 8 ) –> ( 1 2 4 5 8 )  
( 1 2 4 5 8 ) –>  ( 1 2 4 5 8 )  

Third Pass:  
( 1 2 4 5 8 ) –> ( 1 2 4 5 8 )  
( 1 2 4 5 8 ) –> ( 1 2 4 5 8 )  
( 1 2 4 5 8 ) –> ( 1 2 4 5 8 )  
( 1 2 4 5 8 ) –> ( 1 2 4 5 8 )
